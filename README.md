# Task 4 - SQL Joins (Chinook Database)

## Objective
To practice SQL JOIN operations (INNER JOIN, LEFT JOIN) and answer business questions using the Chinook database.

## Tools Used
- DB Browser for SQLite
- Chinook SQLite Database

## Queries Performed
1. INNER JOIN between Invoice and Customer tables.
2. LEFT JOIN to find customers with no orders.
3. JOIN between InvoiceLine and Track to calculate revenue per product.
4. JOIN between Track and Genre to calculate category-wise revenue.
5. Business query using date filter.

## Files Included
- joins_queries.sql : All SQL queries
- joined_output.csv : Final query output
- insights.txt : Business insights

## Outcome
Learned how to combine multiple tables using SQL joins and generate meaningful business insights.
